# Requirements

## Runtime

- Java 23 or later
- No external dependencies — pure Java Swing

## Build

- Gradle 8.13 (wrapper included)
- Java 23 toolchain
- JUnit 5.11.4 for testing

## Features

- 4 robots per match, last alive wins
- User-controlled robot with configurable AI strategies
- 3 built-in AI opponents with distinct behaviors
- Scan rays, projectiles, grenades with area-of-effect
- Obstacle collision and cover mechanics
- Heart pickups for healing
- Persistent win/loss tracking
- Persistent strategy profiling with auto-selection
- Game event logging
