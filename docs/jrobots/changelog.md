# Changelog

## 1.0.0 — Initial version

- Top-down 2D robot battle game with 4 robots
- 3 AI opponents (Aggressive, Sniper, Evasive) using state machine
- User-controlled robot with pluggable strategy system
- 6 scan strategies, 7 speed strategies, 5 targeting strategies
- Strategy profiles persisted to `~/.jrobot_strategy.dat`
- Win counting persisted to `~/.jrobot_wins.dat`
- Game logging to `build/logs/jrobot.log`
- Swing-based rendering with robot skins, scan lines, explosions
